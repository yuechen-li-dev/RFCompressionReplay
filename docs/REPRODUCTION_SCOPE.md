# Reproduction Scope

This repository is an independent reproduction harness for a 2018 RF spectrum-sensing paper.

- Any datasets used by this repository will be regenerated by this harness unless external data is explicitly supplied in a later milestone.
- M0 establishes typed execution, deterministic control flow, artifact generation, and reproducibility plumbing only.
- M0 does **not** claim exact numeric replication of the original paper's unpublished data or results.
- Later milestones may add real detector implementations, regenerated datasets, and richer evaluation logic without changing the M0 scope above.
