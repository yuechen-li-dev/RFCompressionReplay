# Reproduction Scope

This repository is an independent reproduction harness for a 2018 RF spectrum-sensing paper.

- Any datasets used by this repository will be regenerated by this harness unless external data is explicitly supplied in a later milestone.
- M0 established typed execution, deterministic control flow, artifact generation, and reproducibility plumbing.
- M1 adds explicit ED, CAV, and paper-statistic `lzmsa-paper` detector implementations over deterministic synthetic windows.
- M1 does **not** claim exact numeric replication of the original paper's unpublished data or results.
- Later milestones may add richer synthetic datasets, a truer LZMA backend, external data, and broader evaluation logic without changing the scope above.
