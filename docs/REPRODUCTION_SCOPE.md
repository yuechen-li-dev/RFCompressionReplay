# Reproduction Scope

This repository is an independent reproduction harness for a 2018 RF spectrum-sensing paper.

- Any datasets used by this repository will be regenerated by this harness unless external data is explicitly supplied in a later milestone.
- M0 established typed execution, deterministic control flow, artifact generation, and reproducibility plumbing.
- M1 added explicit ED, CAV, and paper-statistic `lzmsa-paper` detector implementations over deterministic synthetic windows.
- M2 adds seeded synthetic benchmark generation for Gaussian noise, a Gaussian-emitter control, and a modest OFDM-like structured source, along with deterministic SNR mixing and consecutive-window Monte Carlo execution.
- The M2 OFDM-like source is **not** LTE and is not claimed to be standards-faithful.
- The current Gaussian-emitter control and OFDM-like source are independent synthetic regenerations aligned with the paper's experimental logic, not claims of exact unpublished paper data regeneration.
- M2 does **not** claim exact numeric replication of the original paper's unpublished data or results.
- Later milestones may add richer synthetic datasets, a truer LZMA backend, external data, broader evaluation logic, and figure reproduction work without changing the scope above.
